Menu Fixo (position: fixed)

Entendendo o Posicionamento CSS

1. Position: Static

Este é o comportamento padrão. O elemento segue o fluxo natural da página, um abaixo do outro.

Sou Static

2. Position: Relative

O elemento é movido em relação à sua posição original, mas o "buraco" que ele ocupava continua lá.

Sou Relative (movi 20px)

3. Position: Absolute

O elemento "voa" e se posiciona em relação ao seu pai que tenha position: relative.

Pai (Relative)
Sou Absolute
